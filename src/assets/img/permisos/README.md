# Imágenes de Permisos

Esta carpeta contiene las imágenes de los vehículos para cada tipo de permiso.

## Estructura de nombres de archivos

Las imágenes deben seguir esta nomenclatura:

- `permiso-a-1.jpg`, `permiso-a-2.jpg` - Para el permiso A
- `permiso-am-1.jpg`, `permiso-am-2.jpg` - Para el permiso AM
- `permiso-a1-1.jpg`, `permiso-a1-2.jpg` - Para el permiso A1
- `permiso-a2-1.jpg`, `permiso-a2-2.jpg` - Para el permiso A2
- `permiso-b-1.jpg`, `permiso-b-2.jpg` - Para el permiso B
- `coche-moderno.jpg` - Imagen de coche moderno para el permiso B
- `permiso-be-1.jpg`, `permiso-be-2.jpg` - Para el permiso B+E
- `permiso-c-1.jpg`, `permiso-c-2.jpg` - Para el permiso C
- `permiso-ce-1.jpg`, `permiso-ce-2.jpg` - Para el permiso C+E
- `permiso-d-1.jpg`, `permiso-d-2.jpg` - Para el permiso D

## Nota

Si una imagen no existe, el componente la ocultará automáticamente sin mostrar errores.
