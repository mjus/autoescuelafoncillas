# Imágenes del Proyecto

## Estructura de Carpetas

Las imágenes deben colocarse en esta carpeta (`src/assets/img/`).

## Imágenes para la página "Nosotros"

Para que las imágenes se muestren correctamente en la página "Nosotros", guarda las fotos del sitio original con estos nombres:

- `nosotros-1.jpg` - Imagen de las instalaciones/oficina
- `nosotros-2.jpg` - Imagen de la pista de aprendizaje  
- `nosotros-3.jpg` - Imagen de los vehículos/flota

## Cómo agregar las imágenes:

1. Descarga las imágenes de https://autoescuelafoncillas.es/nosotros/
2. Guarda las imágenes en esta carpeta con los nombres indicados arriba
3. Las imágenes se mostrarán automáticamente en la página

## Nota

Si las imágenes no están disponibles, se mostrarán placeholders con iconos automáticamente.
